# 📋 Task Manager 
A task management app built with React and supabase
## Features
- Add, edit and delete tasks
- Set task priority (High, Medium, Low)
- Filter tasks by type and priority
- Search tasks
- Dark mode
- Reward system when completing tasks
## Tech Stack
- React
- Supabase
- React Router
- Tabler Icons
## Getting Started 
- npm install
- npm run dev
